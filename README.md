# TodoApp-Handson

シンプルなタスク管理アプリケーションです。

## 概要

ブラウザだけで動作する軽量なTodoアプリです。タスクの追加・編集・削除・完了管理ができ、データはブラウザのLocalStorageに保存されるためサーバー不要で動作します。

## 技術スタック

| 技術 | 用途 |
|------|------|
| **HTML** | アプリケーションの構造・マークアップ |
| **CSS** | スタイリング・レイアウト |
| **JavaScript** | インタラクション・ロジック処理 |
| **LocalStorage** | タスクデータの永続化（ブラウザ内保存） |

## 機能

- タスクの追加
- タスクの編集
- タスクの削除
- 完了/未完了の切り替え

## 使い方

1. リポジトリをクローン
   ```bash
   git clone https://github.com/sou-koshiro/todo-app-handson.git
   cd todo-app-handson
   ```

2. `index.html` をブラウザで開く
   ```bash
   open index.html
   ```

> サーバー不要でブラウザから直接開けます。

## データの保存について

タスクデータはブラウザの **LocalStorage** に保存されます。  
同じブラウザ・同じオリジンでアクセスすればデータが引き継がれます。  
ブラウザのキャッシュをクリアするとデータは削除されます。
